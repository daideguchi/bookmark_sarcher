Book Makuri Search Privacy Policy

Last updated: June 1, 2026

Single purpose

Book Makuri Search helps users search bookmarks saved in their Chrome browser from a compact popup.

Data collection

This extension does not collect, store, sell, or transfer personal data. Bookmark titles and URLs are accessed only inside the user's browser to show local search results.

Permissions

The extension uses the Chrome bookmarks permission only to search the user's saved bookmarks when the user types a search keyword.

Remote code and third parties

The extension does not execute remote code, does not load ads, and does not send bookmark data to third-party services.

Limited use statement

The use of information received from Chrome APIs will adhere to the Chrome Web Store User Data Policy, including the Limited Use requirements.

Contact

For questions about this policy, contact the developer through the GitHub repository: https://github.com/daideguchi/bookmark_sarcher.
